### 안녕하세요, 아래는 제 간단한 이력입니다.

- 웹 디자인 및 퍼블리싱으로 프로그래밍 입문
- Node.js 기반 웹 서비스 개발 회사 취업해서 1년 조금 넘게 근무하고 퇴사
  - Flutter로 WebView 기반 모바일 어플리케이션 구현
  - Telegram Bot을 이용한 알림봇 구현 (서버 헬스 체크 실패시 채팅으로 알림)
  - 복잡한 배포 설정 과정을 쉘 스크립트로 자동화해서 업무 시간을 50퍼센트는 더 확보한 경험 (서버 컨픽, nginx 설정, 사용 가능한 포트 할당, Batch 설정 등 스크립트 한번에 작업하도록 자동화)
- 파이썬 웹 크롤링 프로그램 제작 및 판매 경험

- 원티드 백엔드 인턴십 프리온보딩 교육 이수 (NestJS Backend)
- JSCODE NestJS 교육 과정 이수 (NestJS Backend)
